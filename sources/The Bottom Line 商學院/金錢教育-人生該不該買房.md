# 金錢教育-人生該不該買房

# 從100萬到1000萬

- 持續儲蓄
- 需要開始理解金融市場(金融四大領域)
- 多學、多看、少做，把控制虧損看成最重要的事情
    - 賺多賺少是老天爺決定，但虧損多少是自己決定的
    - 當狀態不好要離開市場，有沒有足夠的耐心取決於有沒有足夠的本金

### 金融四大領域，如何槓桿你的財富

### 銀行 = 信用

信貸不會審核身體狀況，所以如果重大疾病時，申請信貸?

租房比較便宜，那麼租房與買房中間的差額，是使用在哪裡呢?

信用的重點在償債能力，而非絕對債務金額

注重在本業的提升

### 基金 = 委託

委託他人投資就不再關注，是一種對自己資產的不負責任。

### 保險 = Mitigation抵減

需要有要抵減的動機，才會去買保險，所以要找出 key risk

保險要保大不保小，在台灣住院的機率之小，我們保單購買住院險的原因?

財富下限: 如果今天發生某某事，所以家庭經濟出了問題，那大多是本來這個家庭經濟就不太理想，這是回到一開始的家庭經濟條件。

保遠，不保短: 

### 資本市場

投資的增值機會來自風險溢酬

- 主動投資的選擇標的
    1. 可持續性
    2. 可擴張性

4. 回饋社會

找到下一個供需失衡的市場/產業/產品: 爆發性的成長，就是來自於供需失衡。基本上就是10倍。
