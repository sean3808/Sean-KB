| 路徑 | 用途 |

| `_inbox/` | 候選素材（fleeting / imports / voice），待 Sean 判斷 |

| `maps/` | MOC / LYT 導航地圖 |

| `_okf/` | generated OKF v0.1 bundle（可刪除重建） |
