# 訴求不一致時額外 feature 會增加成本但降低價值

## 核心想法
當產品訴求（如折扣訴求 vs 尊榮訴求）與所提供的 feature 不一致時，增加 feature 只會拉高成本，而非提升轉換效果。

## 展開 / 理由
信用卡行銷 AB 測試顯示：「尊榮訴求」搭配「尊榮表格（專人辦理）」效果最佳；「折扣訴求」搭配尊榮表格反而效果差，因為訴求與體驗不搭配。講師以「主打折扣，卻開轎車載你去吃臭豆腐」比喻訴求衝突的荒謬感。在訴求未釐清前盲目加功能，等於花錢降低價值。

> 「專人辦理」很貴；實驗做完你就會知道，這麼貴的 feature 不該對每個人都給，而是要在「訴求一致」的情境下給，才合理。否則就會變成：我這邊主打折扣，卻又開轎車載你去吃臭豆腐；聽起來就是不對。當你還沒把價值訴求釐清，就亂加 feature，其實是在增加成本、降低價值。
